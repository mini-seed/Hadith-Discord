# Hadith-Discord

1. Clone the Repository
```
git clone https://github.com/mini-seed/Hadith-Discord.git
```

```
npm install
```
3. Configure the bot in `config.json`
```json
{
    "PREFIX": "!",
    "TOKEN": "You can get your token from http://discord.com/developers"
}
```
4. start it!
```
npm start
```
